About torth

torth (Typed fORTH) is a experimental programming language, going to be designed to check some conceptions, like for example those:

  • colour usage instead or in combination with some syntax elements
  • building words from words parts
  • delegation of computations to different execution instances
  • rich and easily expandable type system
  • non-english default words for better divide words into parts, for example Esperanto
  • working with multiple IO devices with simple and good looking syntax
  • being very portable, usable an devices in range from 8-bit microcontrollers to powerful server machines
  • no one will say that torth isn't tort (Russian торт - cake, pie - means something good)
